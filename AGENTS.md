# AgentScope.NET 开发规范

## 命名规范

### 中文命名优先
尽可能用中文命名方法名、参数名、变量名、类名、接口名等。

- **方法名**: 使用中文描述方法功能（如：获取用户列表、发送消息、处理请求）
- **参数名**: 使用中文描述参数含义（如：用户名称、消息内容、配置选项）
- **变量名**: 使用中文描述变量用途（如：会话列表、聊天记录、工具映射）
- **类名/接口名**: 使用中文描述类/接口职责（如：消息服务、工具管理器、对话处理器）

### 示例

```csharp
// 类名使用中文
public class 消息服务 { }
public interface I工具管理器 { }

// 方法名使用中文
public async Task<消息> 发送消息Async(消息内容, 接收者) { }

// 变量名使用中文
var 用户列表 = await 获取用户Async();
var 聊天记录 = new List<消息>();
```
